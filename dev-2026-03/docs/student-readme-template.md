# 项目名称

## 项目简介

这是一个使用 Python 和 pandas 完成的电商零售数据清洗项目。

## 我完成的内容

- 读取原始数据
- 删除重复记录
- 删除 Customer ID 为空的记录
- 删除 Quantity <= 0 的记录
- 删除 Price <= 0 的记录
- 处理 InvoiceDate
- 导出清洗结果
- 输出统计信息

## 我的实现思路

### 1. 数据读取

我先使用 pandas 读取原始数据文件。

### 2. 数据清洗

我按题目要求完成了基础清洗。

### 3. 导出结果

我把清洗后的数据导出到了 output 目录。

### 4. 统计信息

我输出了原始行数、清洗后行数和删除记录数量。

## 运行方式

1. 创建并激活虚拟环境
2. 安装依赖
3. 下载并准备数据
4. 运行 `python main.py`
