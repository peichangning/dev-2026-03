# 评分细则（面向 agent）

总分 100 分。

## 评分原则

1. 这道题面向初学者，不按数据分析岗位高级标准打分。
2. 优先看程序能不能运行、清洗任务有没有完成。
3. 只要逻辑基本正确，就不要因为代码不够优雅而给过低分。
4. 如果 README 能清楚说明做了什么，可以适当正向考虑。

## 一、数据读取与程序可运行性，20 分
- 程序可以正常运行，10 分
- 能正确读取数据文件，10 分

## 二、基础数据清洗完成度，40 分
- 删除重复数据，10 分
- 删除 Customer ID 为空记录，10 分
- 删除 Quantity <= 0 的记录，10 分
- 删除 Price <= 0 的记录，10 分

## 三、日期处理与结果导出，15 分
- InvoiceDate 正确转成日期时间，10 分
- 成功导出清洗后结果文件，5 分

## 四、清洗统计信息，15 分
- 能输出基础统计结果，10 分
- 统计结果基本正确，5 分

## 五、代码质量与 README，10 分
- 代码结构基本清楚，5 分
- README 能说明做了什么，5 分
